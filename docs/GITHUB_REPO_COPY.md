## Pinned Repo Intro

### 中文版

```text
Job Hunter Skill 是一个面向真实求职场景的自动化投递工具。它基于 DrissionPage 和本地浏览器 CDP 接管，支持 Boss直聘 与 实习僧，并把简历解析、JD 匹配评分、安全演练、正式投递和结构化日志串成了一条完整链路。这个项目当前是 alpha 版本，目标不是“大而全”，而是先把真实可跑通、可扩展、可开源协作的基础打稳。
```

### English Version

```text
Job Hunter Skill is an automation tool for real-world job application workflows. It uses DrissionPage with local CDP browser takeover and currently supports Boss直聘 and Shixiseng, connecting resume parsing, JD scoring, rehearsal mode, real apply mode, and structured logging into one workflow. The project is currently in alpha and is focused on being practical, extensible, and open-source friendly rather than trying to support everything at once.
```
