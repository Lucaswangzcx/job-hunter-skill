# GitHub 仓库文案

这份文案用于 GitHub 仓库的 `About`、`Topics`、置顶简介和首页短介绍。

## 仓库名

建议保留：

```text
job-hunter-skill
```

展示名建议：

```text
Job Hunter Skill
```

## GitHub About

GitHub 右侧 `About` 区域建议直接用这句中文：

```text
一个基于 DrissionPage 和 CDP 接管的求职自动化工具，支持 Boss直聘 和 实习僧，包含简历匹配、安全演练、正式投递和结构化日志。
```

如果你想更强调“开源”和“实用”，可以用这个版本：

```text
面向中国大学生和求职者的开源求职自动化工具，基于 DrissionPage + CDP 浏览器接管，支持 Boss直聘 和 实习僧，并提供简历匹配、演练投递和日志分析。
```

## 置顶简介

```text
Job Hunter Skill 是一个面向真实求职场景的自动化投递工具。它基于 DrissionPage 和本地浏览器 CDP 接管，支持 Boss直聘 与 实习僧，并把简历解析、JD 匹配评分、安全演练、正式投递和结构化日志串成了一条完整链路。这个项目当前是 alpha 版本，目标不是“大而全”，而是先把真实可跑通、可扩展、可开源协作的基础打稳。
```

## 首页短介绍

```text
一个真正面向求职投递场景的自动化工具：手动登录浏览器、脚本接管会话、匹配 JD、达到阈值后再投递，优先保证可控、可演练、可扩展。
```
