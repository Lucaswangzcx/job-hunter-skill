# 行为准则

## 我们的承诺

我们希望 `Job Hunter Skill` 成为一个对用户和贡献者都友好的项目。无论你是刚入门还是经验丰富，我们都希望你能在这里感到被尊重、被认真对待。

## 我们期待的行为

下面这些行为有助于营造积极的协作氛围：

- 尊重他人，表达清楚
- 提供具体、可执行的建议
- 假设对方是善意的，但也要诚实指出问题
- 尊重隐私与安全

下面这些行为是不被接受的：

- 人身攻击、骚扰、歧视
- 公开别人的简历、账号、密钥、浏览器数据
- 恶意带节奏或反复争吵
- 不负责任地传播漏洞或绕过方式

## 执行

维护者可以根据情况移除评论、Issue、PR 或贡献权限。

## 适用范围

本准则适用于：

- Issue
- PR
- Discussions
- Release Notes
- 其他项目沟通渠道
