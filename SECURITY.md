# 安全策略

## 支持版本

我们只对最新的发布分支提供安全修复支持。

## 如何报告漏洞

请不要直接用公开 Issue 报告安全敏感问题。

请私下联系维护者，并尽量说明：

- 受影响版本
- 复现步骤
- 风险影响
- 你认为可以怎么缓解

如果问题涉及以下内容，请在标题里写清楚：

- 凭据泄露
- 非预期的浏览器接管
- 不安全的本地文件写入
- LLM 密钥暴露

## 范围说明

这个项目会接触：

- 真实网站
- 本地已登录的浏览器
- 个人简历
- 本地日志

请不要在公开讨论里贴出：

- 简历全文
- 浏览器 Profile
- API Key
- 可识别个人身份的日志
