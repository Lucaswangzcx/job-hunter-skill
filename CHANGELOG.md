# 更新日志

- 首个公开 `alpha` 版本。
- 支持 `Boss直聘` 和 `实习僧`。
- 只使用 `DrissionPage + CDP` 浏览器接管模式。

- 增加 `pyproject.toml`、`requirements.txt`、示例配置、License、CI 和贡献文档。
